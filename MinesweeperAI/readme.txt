Detailed Instructions on How to Run the Code:


To run the Minesweeper AI code, follow these steps:

1. Ensure that Python is installed on your system.
2. Download the code files (`player.py`, `move.py`).
3. Open a terminal or command prompt.
4. Navigate to the directory containing the code files.
5. Run the command `python Mines.py` to execute the Minesweeper AI.
